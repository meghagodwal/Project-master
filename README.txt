Random User API with React

This is a basic React project which utilises the Random User API. Data is pulled from the API and output into a lightly styled list on the page.

Requirements:
Download the Node.js from the official Node website, link given below-

https://nodejs.org/en/download/

Setting up a ReactJS project

Step 1- Install Create React App by following command - npm install -g create-react-app
Step 2 - Create a react using following command - create-react-app #projectname

How to use :

Step 1:

git clone https://github.com/meghagodwal/Project-master

Step 2:

npm install

Step 3:

npm start

Step 3:

Visit http://localhost:3000/
